Получение всех фильмов.

SELECT
    *
FROM film

Получение всех пользователей.

SELECT
    *
FROM user

Получение N наиболее популярных фильмов.

SELECT
    COUNT(user_id) AS likes
FROM film_like
GROUP BY film_id
ORDER BY COUNT(user_id) DESC
LIMIT = N
